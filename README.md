# ATTARIK — The Art of Scent

Premium luxury fragrance website built with Node.js, Express, and vanilla HTML/CSS/JS.

## Quick Start

```bash
npm install
npm start
```

Visit `http://localhost:3000`

## Deployment on Render

1. Push this repo to GitHub
2. Connect to Render as a Web Service
3. Build command: `npm install`
4. Start command: `node server.js`

## API Endpoints

- `GET /api/health` — Health check
- `POST /api/contact` — Contact form submission
- `POST /api/newsletter` — Newsletter subscription
- `GET /api/admin/contacts?key=attarik2024` — View contacts (admin)
- `GET /api/admin/subscribers?key=attarik2024` — View subscribers (admin)
